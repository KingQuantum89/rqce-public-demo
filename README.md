RQCE Public Demonstration



This repository contains a simplified public demonstration environment used to illustrate behavioral properties of a recursive optimization architecture.



Purpose

This release is intended for:

\- reproducibility testing

\- independent verification

\- academic inspection



It does not contain proprietary implementation layers or full architecture internals.



Requirements

Python 3.10+



Install dependencies:

pip install -r requirements.txt



Run demo:

python demo.py



Notes

This is a minimal validation environment, not the full system.
